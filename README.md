# webOS
Stardance - webOS

This is my web-based OS project for the Stardance Challange.

------------------------------

My OS is called Brainrot OS. It features a meme/brainrot themed boot animation when opening the webpage. After the system "boots", there are three applications on the desktop, but also a working clock and a date display. The setting feature doesn't work yet. All of the applications open a draggable window. The first one, called brainrot.exe just opens a window which says "you are brainrotted bro". The notes app is a basic note taking app. The third app, called "do not open" is a minigame: if the user opens it, a window with a big red button pops up, if the user presses the button, a few meme themed errors pop up on the screen, and the user has to exit each of them manually.
